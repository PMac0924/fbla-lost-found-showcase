# Waukee Lost & Found

A secure, school-managed lost and found web application built for FBLA Website Coding & Development.

## Overview

Waukee Lost & Found is a database-driven website designed to help schools manage lost and found items in a more organized way. Instead of relying only on a physical box near the office, the system allows students to report lost items, submit found items, view potential matches, and request claims through an online platform.

This project was created for FBLA Website Coding & Development and placed 6th nationally.

## Problem

Many schools handle lost and found items through a basic office box or informal announcements. While that can work for simple items, it becomes harder when students need to search for missing items, verify ownership, track claims, or organize submissions over time.

## Solution

Waukee Lost & Found creates a more structured system for managing lost and found items. It gives students a simple way to submit information and gives administrators tools to review, organize, and manage item claims.

## Key Features
